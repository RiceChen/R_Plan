## R plan Little Box
- 《Little Box》是一个体积极小，功能齐全的桌面小盒子。功能列表如下：

| 模块 | 功能 |
|-----|------|
| BK7252U WIFI模块 | 作为主控（主频180MHz）及wifi联网功能 |
| BLE | 辅助配网 |
| 摄像头OV7670 | 图像识别，拍照 |
| LCD | 图片显示，时间显示，环境温湿度显示 |
| SHT3X | 获取环境温湿度 |
| MIC | 录音，语音识别控制，声波配网， |
| 喇叭 | 录音播放，音乐播放 |
| SD card | 保存图片，存放音乐，数据存储 |
